# 🤖 CodeAlpha FAQ Chatbot

## 📌 Project Description
This is an AI-powered FAQ Chatbot developed using Python, Streamlit, and Scikit-learn. It answers user questions by finding the most similar question from a predefined FAQ dataset using TF-IDF Vectorization and Cosine Similarity.

## 🚀 Features
- Interactive Streamlit interface
- Text preprocessing
- TF-IDF Vectorizer
- Cosine Similarity matching
- Fast FAQ retrieval

## 🛠 Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## ▶️ Run the Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
